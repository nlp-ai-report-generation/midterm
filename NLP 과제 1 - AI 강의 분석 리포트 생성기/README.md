### 강의 스크립트 구성

- **1주차** (2026-02-02 ~ 02-06): 객체지향 디자인패턴, HTML/CSS, JavaScript
- **2주차** (2026-02-09 ~ 02-13): React.js, 컴포넌트 스타일링
- **3주차** (2026-02-23 ~ 02-27): React Query, HTTP 통신 개요 및 웹 작동 방식

### 강의 품질 평가 체크리스트 구성 (v3)

체크리스트는 5개 카테고리, 18개 항목으로 구성되어 있으며, LLM 프롬프트 설계의 기준으로 활용합니다.

| 카테고리 | 항목 수 | 주요 평가 항목 |
|---|---|---|
| 1. 언어 표현 품질 | 3개 | 불필요한 반복 표현, 발화 완결성, 언어 일관성 |
| 2. 강의 도입 및 구조 | 5개 | 학습 목표 안내, 전날 복습 연계, 설명 순서, 핵심 강조, 마무리 요약 |
| 3. 개념 설명 명확성 | 4개 | 개념 정의, 비유/예시 활용, 선행 개념 확인, 발화 속도 적절성 |
| 4. 예시 및 실습 연계 | 3개 | 예시 적절성, 실습 연계, 오류 대응 |
| 5. 수강생 상호작용 | 3개 | 이해 확인 질문, 참여 유도, 질문 응답 충분성 |

> 💡 각 항목을 LLM 프롬프트로 변환하여 자동 분석 시스템을 구축하는 것이 이 과제의 핵심입니다.
