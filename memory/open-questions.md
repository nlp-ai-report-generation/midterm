# 열린 질문

## 제품 및 분석 방향

### 어떤 분석 항목을 MVP 우선순위로 둘 것인가

- 상태: 미결정
- 영향: 프로토타입 범위와 리포트 구조가 달라진다

### 점수 체계는 어떻게 설계할 것인가

- 상태: 미결정
- 영향: 정량 지표, 비교 분석, 리포트 신뢰도에 직접 영향

### 규칙 기반과 LLM 기반의 역할 분담은 어떻게 할 것인가

- 상태: 미결정
- 영향: 구현 난이도, 비용, 일관성, 정확도에 영향

## 산출물 및 데모

### 최종 산출물의 중심은 대시보드인가 리포트 파일인가

- 상태: 미결정
- 영향: UI 범위와 개발 리소스 배분이 달라진다

### 중간평가 시연에서 무엇을 최소 기능으로 보여줄 것인가

- 상태: 미결정
- 영향: 2주차까지의 개발 우선순위가 달라진다

## 구현 및 운영

### 기술 스택은 어떤 조합으로 확정할 것인가

- 상태: 미결정
- 영향: 분석 파이프라인, 배포, 문서 생성 방식이 달라진다

### 강사별 비교 분석까지 MVP에 포함할 것인가

- 상태: 미결정
- 영향: 데이터 모델과 리포트 범위가 커질 수 있다
