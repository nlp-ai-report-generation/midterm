# 데이터 및 문서 인벤토리

## 루트 문서

### `README.md`

- 용도: 프로젝트 개요, 평가 기준, 제공 데이터 요약
- 신뢰도: 높음
- 비고: 외부 설명과 내부 작업 기준의 출발점

### `docs/decision-points.md`

- 용도: 팀이 논의해야 할 핵심 의사결정 목록
- 신뢰도: 높음
- 비고: 아직 결정되지 않은 질문이 많다

### `AGENTS.md`

- 용도: 저장소 공용 작업 운영 규칙
- 신뢰도: 높음
- 비고: 에이전트는 이 문서를 실무 기준으로 따른다

## 과제 폴더 자료

### `NLP 과제 1 - AI 강의 분석 리포트 생성기/README.md`

- 용도: 과제 상세 설명, 주차별 마일스톤, 제출물 기준
- 신뢰도: 높음
- 비고: 기술 스택은 권장안이며 확정안은 아니다

### `강의 스크립트/*.txt`

- 용도: 강의 분석의 핵심 입력 데이터
- 신뢰도: 중간
- 비고: STT 데이터이므로 오인식과 화자 혼합 가능성 존재

### `강의 메타데이터.csv`

- 용도: 강의 배경 정보와 세션 구분
- 신뢰도: 높음
- 비고: 집계, 정렬, 비교 분석 기준 데이터

### `강의 품질 기준.pdf`

- 용도: 원본 체크리스트 문서
- 신뢰도: 높음
- 비고: 원문 보존용 자료

### `강의 품질 기준.md`

- 용도: 체크리스트를 읽기 쉬운 텍스트로 정리한 파생 문서
- 신뢰도: 높음
- 비고: 분석 항목 매핑 시 우선 참조 권장

## 사용 원칙

- 원본 데이터를 새 문서에 통째로 복제하지 않는다.
- 보고용 문서에는 필요한 최소 인용과 요약만 사용한다.
- 근거를 남길 때는 파일 경로나 세션 단위를 함께 적는다.
