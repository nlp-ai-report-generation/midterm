| 사용 비권장 | hop_90pct — 3분 오버랩으로 커버리지 부족, 일부 날짜 결과 누락 |

**종합 권장**: `hop_50pct (15분, 50% overlap)` 설정이 전체 커버리지, 상호작용 포착, 맥락 연속성 모든 면에서 가장 안정적인 결과를 제공한다. API 비용이 제약 조건일 경우 hop_75pct를 차선으로 고려할 수 있으나, hop_90pct는 커버리지 손실이 크므로 실제 평가에는 부적합하다.
