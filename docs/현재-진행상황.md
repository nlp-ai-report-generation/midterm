## 진행 중인 작업

### 하네스 정교화

평가 기준에서 모호한 표현을 찾아 수치 기반 행동 지표로 전환하고 있다.

| 항목 | 현재 | 변경 후 | 상태 |
|------|------|--------|:----:|
| 3.2 비유/예시 활용 | "적절한 비유" | 구조적 대응 여부 + 빈도 숫자 | ✅ |
| 5.2 이해 확인 질문 | "자주 함" | 30분당 3회 이상 + 내용 확인형 50% | 🔄 |
| 5.3 참여 유도 | "적극적으로" | 참여 유형 분류 + 빈도 | 🔄 |
| 2.3 설명 순서 | "논리적" | 선수→후속 개념 위반 횟수 | ⬜ |
| 3.3 선행 개념 확인 | "가끔" | 심화 전환 지점 대비 확인 비율 | ⬜ |

정교화 전후 ICC를 비교하여 효과를 검증할 예정이다.
