## 실험 2. 청크 크기 영향 검정

### 연구 질문

> 강의 스크립트를 30분 단위 vs 15분 단위로 나눴을 때, 평가 결과에 차이가 있나요?

### 실험 설계

| 항목 | 조건 A | 조건 B |
|------|:------:|:------:|
| **청크 크기** | **30분** | **15분** |
| 오버랩 | 5분 | 5분 |
| 모델 | GPT-4o-mini | GPT-4o-mini |
| Temperature | 0.1 | 0.1 |
| 보정기 | OFF | OFF |
| 실험 ID | `edca9e189002` | `0910631aee57` |

### 가설

- **H₀:** 30분 청크와 15분 청크의 평가 점수 평균이 같아요 (μ₃₀ - μ₁₅ = 0)
- **H₁:** 30분 청크와 15분 청크의 평가 점수 평균이 달라요 (μ₃₀ - μ₁₅ ≠ 0)

### 결과

### 해석

**H₀ 기각.** 30분 청크와 15분 청크는 **통계적으로 유의미한 차이**를 만들어요.

- 30분 청크가 15분 청크보다 평균 **+0.21점** 높아요
- 15개 강의 중 **13개(87%)**에서 30분 청크 점수가 더 높았어요
- Cohen's d = 1.14로 **큰 효과 크기**예요

**왜 이런 차이가 생길까요?**

30분 청크는 더 넓은 맥락을 담고 있어서, LLM이 강의의 흐름과 구조를 더 잘 파악할 수 있어요. 15분 청크는 맥락이 좁아서 "학습 목표 안내 → 본문 설명 → 마무리" 같은 구조적 요소를 놓칠 가능성이 있어요.

**실무 시사점:**
- 평가 시 **30분 청크를 기본값으로 유지**하는 게 좋아요
- 15분 청크는 점수를 보수적으로(낮게) 매기는 경향이 있으니, 벤치마크 비교 시 청크 크기를 통일해야 해요
